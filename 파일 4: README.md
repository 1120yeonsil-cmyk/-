## 🎯 프로젝트 개요

**Re:Pet**은 반려동물을 잃은 사람들을 위한 감정적 지원 앱입니다. 사용자는 반려동물의 사진과 원하는 행동을 입력하면, 마치 생전의 모습처럼 보이는 장면을 감상할 수 있습니다. 나아가 먹이주기, 목욕시키기, 응원 메시지 남기기 등의 돌봄 활동을 통해 반려동물을 계속 추모할 수 있습니다.

## ✨ 주요 기능

### 1. 📸 사진 업로드
- 반려동물의 사진을 간편하게 업로드
- 실시간 미리보기 제공
- JPG, PNG, GIF 형식 지원

### 2. 🎬 프롬프트 입력 및 추억 생성
- 원하는 행동을 텍스트로 입력
- "추억 생성하기" 버튼으로 즉시 결과 확인
- 2초의 로딩 애니메이션 제공

### 3. 🖼️ 결과 화면
- 생성된 추억을 아름다운 카드 형태로 표시
- 감정적인 메시지 함께 제공

### 4. 💕 돌봄 기능
- **🍖 먹이주기**: 반려동물에게 음식을 줄 수 있음
- **🛁 목욕시키기**: 반려동물을 깨끗이 해줄 수 있음
- **💬 응원 메시지**: 반려동물에게 마음을 전할 수 있음

### 5. 🎨 감성적 UI/UX
- 따뜻한 색상 팔레트 (핑크, 블루, 베이지)
- 부드러운 애니메이션과 전환 효과
- 모바일 반응형 디자인
